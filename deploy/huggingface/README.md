# Continuum on Hugging Face Spaces

This Space runs the Continuum remote MCP server, so you can add it to Claude web, Grok, or ChatGPT
as a custom connector.

Your MCP URL is `https://rahultwte-continuum-mcp.hf.space/mcp`.

The Space contains the Continuum source (`pyproject.toml` and `src`), this README, and the
Dockerfile. Set your secrets in the Space settings under Variables and secrets.
